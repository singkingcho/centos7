# 二、环境准备

要想学好Linux，需要有环境作为练习，Linux发行版非常多，但是你学习了一个之后，其它的学习都是非常简单的，本次使用centos7作为讲解。

为了装上centos7我们没有必要去买个真的机器来装，我们使用虚拟机来安装即可，常见的虚拟机软件

- VMware
- Virtual box



本次选用box，如果跟随学习，最好软件保持一致。

最后环境约束说明：

 - win10(非必须)
 - virtual box (必须，版本随意)
 - centos7(必须)
